# TP4 React Native - Navigation

## Description
Application React Native démontrant la navigation par onglets et par pile.

## Fonctionnalités
- Navigation par onglets (Tab Navigation)
- Navigation par pile (Stack Navigation)
- AppBar personnalisée
- SafeAreaView pour les zones sûres
- 3 écrans : Accueil, Détails, Paramètres

## Technologies
- React Native
- React Navigation
- Expo

## Installation
```bash
npm install
npx expo start
